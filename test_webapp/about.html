📖 About This Application

v2.0 - Testing Phase

このアプリケーションはScreenshot Manager for WSLのテスト用に作成されました。

🔧 主な機能:

🔍 自動検知

Webアプリの起動を自動検知

📸 スクリーンショット

複数解像度でのキャプチャ

🔄 ページ巡回

設定されたページの自動巡回

📊 レポート生成

HTMLレポートの自動生成

  • フロントエンド: Vanilla HTML/CSS/JavaScript
  • 監視: Python + Port Monitoring
  • キャプチャ: Playwright + Chromium
  • レポート: HTML生成
